# script-language
Welcome to the "script" program language

This language main objective is to make syntax more simple

Basic printing:
To print, just use `console.write("hello, world!")` and if you want styling, use:

```javascript
console.write("red text!" style=[text-color: red, outline-size: 5px, outline-color: black])
```

read the full documentation here: https://script-langs.github.io/script-language/
