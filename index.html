CONSOLE

Console is usually used for console controls, like console.log is used to make console create a log about objects or items


write also used for console, combine it to make a simple printing, example:

console.write("Hello World")

console also can be combined with warn to make a warning with a red text that simply used to warn you or people that are using the code, used for warning situations

console.warn("warning! one of the code is error!")

to make an input, simply use input to create an input text, put it inside of a console.write, example:

console.write("your name:" + input)

FUNCTIONS AND CODES

Functions is mainly used to make a list of code and call it in the same time, when it was called it will run all the code inside that function


to create a function, use function block to make it, example:

function myFunc {
  // your code here
}

and call it using call yourfunction(), change yourfunction() to your own function, for example:

function myFunc {
  console.write("hello from my function")
}
        
call myFunc()

for codes or keywords there are so many keywords or codes in the script language, each keywords has it owns functions, lets start with simple codes/keywords like string[]


string[] is used inside a function, its used to make function code run from top to bottom using string[tp-bm], tp means top, bm means bottom, md means middle


but if you want it to stop in the middle or stop it on a specific line, you can use string[stop]


another keywords is if, as you know, if is used for some situation like if 1 < 2 then {...}, and in here, it uses brackets, like example:

  console.write("1 is less than 2")

Loops also can be looped for whatever time you want using loop repeat(3) {...}, for simple example of loop:

 // your code here, for example:
